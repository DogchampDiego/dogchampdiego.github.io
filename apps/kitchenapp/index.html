WIYK - What's in your Kitchen?

Overview

WIYK helps you manage your kitchen devices, track purchases, and create presets for easy cooking and food preparation. This application streamlines your kitchen workflow, helping you save time and improve efficiency.

Features

  • Device Management: Track and manage all your kitchen devices
  • Purchase Statistics: View detailed statistics about your kitchen purchases
  • Preset Creation: Save and apply common cooking configurations
  • Multi-language Support: Available in multiple languages for global users
  • Intuitive Interface: User-friendly design for easy navigation

Technologies Used

  • Flutter
  • Dart
  • Hive
  • Provider
  • Firebase

Architecture

The app follows a clean architecture pattern with:

  • Providers: For state management
  • Models: Data structures
  • Screens: UI components
  • Widgets: Reusable UI elements
  • Services: Business logic and external interactions
